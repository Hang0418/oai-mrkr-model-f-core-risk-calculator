# AgingResearchBiobank / MOST Request Text

## Project Title

Longitudinal pain trajectories and imaging structural features for predicting total knee arthroplasty risk in knee osteoarthritis: model development in OAI and external validation in MOST.

## Study / Resource Requested

Multicenter Osteoarthritis Study (MOST).

## Requested Materials

Study datasets and documentation for MOST, including:

- Clinical and questionnaire datasets.
- Data dictionaries, dataset descriptions, measurement lists, and variable guides.
- Knee pain, WOMAC/KOOS or comparable symptom and function variables.
- Knee injury, knee surgery, medication, treatment, and comorbidity variables.
- TKA/KR/knee replacement outcome variables and follow-up timing.
- Knee radiograph scoring variables, including KL grade, JSN, osteophyte, and related structural progression measures.
- Documentation needed to harmonize MOST variables with OAI.

Raw MRI/X-ray image files and biospecimens are not requested for the first-stage analysis unless required later through an approved amendment.

## Research Purpose

This project aims to externally validate longitudinal prediction models for clinically important outcomes in knee osteoarthritis. The Osteoarthritis Initiative (OAI) will be used as the model development cohort, and the Multicenter Osteoarthritis Study (MOST) will be used as an independent external validation cohort. The primary outcome is incident target-knee total knee arthroplasty or knee replacement during follow-up. Secondary outcomes include radiographic progression, joint space narrowing progression, worsening pain, and functional decline.

## Scientific Rationale

Knee osteoarthritis is a major cause of chronic pain, disability, and eventual joint replacement. Existing risk prediction studies often use baseline symptoms or imaging features alone. This study will evaluate whether longitudinal pain trajectories and early structural imaging features improve prediction of future knee replacement risk. External validation in MOST is essential to assess whether prediction models developed in OAI generalize to an independent population.

## Analysis Plan

The analysis unit will be the knee level, with person-level sensitivity analyses. OAI and MOST visits will be aligned using landmark windows, with OAI baseline plus 24 months used for model development and MOST baseline plus 30 months used for external validation. Pain and function measures will be harmonized across visits and transformed so that higher values represent worse symptoms or function. Radiographic variables such as KL grade, JSN, and osteophytes will be harmonized where possible.

The primary model will predict target-knee TKA/KR after the landmark visit. Cox proportional hazards models will be used as interpretable baseline models. If sample size and software environment permit, random survival forests or gradient boosting survival models will be explored as secondary machine-learning models. Model performance will be assessed using C-index, time-dependent AUROC, Brier score, calibration, and risk stratification. Results will be reported in aggregate only.

## Data Security / Confidentiality

All approved data will be stored in a secure local research directory with access restricted to the project investigator/team. Raw data will be preserved unchanged, and derived analysis files will be stored separately. The data will be used only for non-commercial scientific research and educational purposes. No attempt will be made to identify or contact individual participants. No individual-level data will be redistributed. Results will be reported only in aggregate form, following all AgingResearchBiobank and MOST data use terms and citation requirements.
